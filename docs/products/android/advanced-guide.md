## Advanced Configuration

### Network Setup
```yaml
TCP Connection:
  IP: 192.168.1.100
  Port: 502 (or custom)
  Timeout: 3000ms
  Protocol: TCP/RTU/ASCII
```

### Serial Configuration  
```yaml
USB-OTG Setup:
  Adapter: USB-to-RS485
  Baud Rate: 9600-115200
  Data Bits: 7/8
  Parity: None/Even/Odd
  Protocol: RTU/ASCII
```
