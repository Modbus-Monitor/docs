## Introduction

The **Modbus Monitor XPF** application is a versatile tool for facilitating communication between a Windows-based Personal Computer and Modbus Client or Server devices. It can be downloaded from the [Microsoft Store](https://apps.microsoft.com/detail/9pg862wl5hsm) or [Download](https://quantumbitsolutions.com/purchase/) Page. This application offers a trial version, allowing users to explore its features before committing to a full license.

One of the standout features of Modbus Monitor XPF is its **dual functionality**, encompassing both a **Modbus Client** and a **Modbus Server** within a single application. This dual capability, combined with support for both **Modbus Serial** and **Modbus TCP** protocols, makes it an invaluable tool for various Modbus communication needs.

### Key Features

- **Dual Mode Operation**: Run both Client and Server simultaneously
- **Protocol Support**: Modbus TCP/IP, Serial RTU, Serial ASCII, and RTU/ASCII over TCP
- **Real-time Monitoring**: Live data visualization from PLCs, drives, meters, and other Modbus devices
- **Device Simulation**: Test equipment without physical hardware
- **Advanced Scripting**: Automation capabilities for licensed users

## Navigating the Application

### Main Application Window

The Modbus Monitor XPF interface is organized into several key sections:

![Main Window Layout](https://quantumbitsolutions.com/wp-content/uploads/2024/09/Main-Window.webp){ .screenshot-shadow }

#### Window Components

1. **Quick Access Toolbar**
   - Located at the top of the window
   - Includes common actions: Save, Undo, Redo
   - Customizable for frequently used functions

2. **Ribbon Tabs**
   - **File**: Document operations and settings
   - **Home**: Primary tools and device management
   - **Client**: Modbus client configuration and operations
   - **Server**: Modbus server setup and simulation

3. **Command Sections**
   - **Device Management**: Add Device, Delete Device, Offset ID
   - **Data Operations**: Read, Write, Monitor functions
   - **Analysis Tools**: Scanner, Logger, Evaluator

4. **Dialog Launchers**
   - Small arrows in ribbon sections
   - Open detailed configuration dialogs
   - Access advanced settings

5. **Window Portal Area**
   - **Monitor Points**: Default view showing configured data points
   - **Log Window**: Historical data and events (toggle with Log button)
   - **Web Files**: Online resources and maps (toggle with Online button)

### Ribbon Interface Overview

The ribbon interface provides organized access to all application features:

#### File Tab
- **New/Open/Save**: Document operations
- **Import/Export**: Configuration file management  
- **License**: Activation and registration
- **Options**: Global application settings
- **Recent Files**: Quick access to previous configurations

#### Home Tab
- **Device Management**: Add, remove, configure Modbus devices
- **Connection Control**: Connect, disconnect, test communications
- **Data Operations**: Manual read/write operations
- **View Options**: Customize display and layouts

#### Client Tab
- **Device Configuration**: Set up Modbus client connections
- **Polling Setup**: Configure automatic data collection
- **Data Mapping**: Define register layouts and scaling
- **Advanced Options**: Timeout settings, retry logic

#### Server Tab  
- **Server Setup**: Configure Modbus server simulation
- **Data Sources**: Define simulated data points
- **Response Configuration**: Set up server behaviors
- **Testing Tools**: Validate server operations
